# 产品化 
使用系统中创建各种 prompt,知识库,tools等组件组合成一个应用(app)对外部进入对话与输出.
基础能力如下:
* 集成多个大模型接口
* prompt (提示词)
* RAG (知识库)
* tools (MCP)
* agent (自定义工作流)
* 应用对话输入与流式输出
* 语音输入
* 对话语音输出
* 应用暴露API供外部使用
* Coding
